# 🏛️ Prime Icon Web Platform
A high-performance multi-page website built with Tailwind CSS v4 and Vanilla JS for Prime Icon.
## 🚀 Tech Stack
- **Styling:** Tailwind CSS v4
- **Scripting:** Vanilla JavaScript (ES6 Modules)
- **Design System:** High-contrast Brutalism
- **Data:** JSON-based content management

## 📂 Project Structure
- `src/css/`: Modular CSS with Tailwind layers.
- `src/js/pages/`: Page-specific logic (e.g., Activity Board).
- `src/js/data.json`: Centralized JSON for dynamic content.

## 🛠️ Local Setup
1. Clone the repo: `git clone [your-link]`
2. Open in VS Code.
3. Start the **Live Server** extension.
4. Enjoy the smooth animations!

## ✨ Key Features
- Dynamic Activity Board with IntersectionObserver.
- Custom Brutalist Hover-Lift components.
- Modular JS architecture for fast loading.
